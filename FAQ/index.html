The VPS that runs the proxy no longer exists. All existing domains that have been proxied will no longer work. We might add it back in the future. I want to try to run the proxy host on my Raspberry Pi so it cost me nothing but I need to look into what the downsides are.

What is illegal to host here?

You cannot host the following: web scrapers, PteroVM, proxies, bot rate limiters, RAM abusers, lavalink servers, resource clogger, bots that purposely use too much RAM (looking at the folks using V3 Aoi.js), miners, servers attempting root access, etc. If you don't know if your server is potentially a violation of this, open a ticket.

Why is lavalink not allowed?

Because it takes too much resources.

Why is my server stuck on installing?

Most common reason because the node is full.

If not ask support through via ticket and wait until help arrives!
